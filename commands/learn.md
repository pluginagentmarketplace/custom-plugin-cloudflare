# /learn - Start Your Learning Journey

This command guides you through selecting a learning path and creating a personalized roadmap.

## How to Use

```
/learn
```

## What It Does

1. **Role Selection** - Choose from 65 different roles across 8 categories:
   - Core Development (Frontend, Backend, Full Stack, DevOps)
   - Languages & Frameworks (Python, JavaScript, Java, Go, Rust, etc.)
   - Mobile & Game Development (iOS, Android, Flutter, Unity, Unreal)
   - Data & AI (Data Engineer, ML Engineer, AI Engineer, Data Analyst)
   - Cloud & Infrastructure (AWS, Kubernetes, Terraform, DevOps)
   - Architecture & Security (System Design, Security, API Design)
   - Emerging Tech & Leadership (Prompt Engineering, Product Manager, Engineering Manager)

2. **Customization** - Answer questions about:
   - Your current experience level
   - Time commitment (hours per week)
   - Learning style preferences
   - Project interests

3. **Personalized Roadmap** - Get:
   - Step-by-step learning path (weeks/months timeline)
   - Recommended resources for each step
   - Project ideas at each level
   - Expected timeline to job readiness

4. **Agent Assignment** - Get paired with relevant agent(s) for:
   - Detailed guidance on each step
   - Technology stack recommendations
   - Answer specific questions
   - Project feedback and suggestions

## Example Paths

### Frontend Developer Path (3-6 months)
- Week 1-2: HTML, CSS, JavaScript fundamentals
- Week 3-4: JavaScript ES6+ and DOM manipulation
- Week 5-6: React basics and component architecture
- Week 7-8: State management (Zustand or Redux)
- Week 9-10: TypeScript and testing
- Week 11-12: Deploy and build portfolio

### Backend Developer Path (6-12 months)
- Month 1: Choose language (Python/JavaScript/Java)
- Month 1-2: Language fundamentals and OOP
- Month 2-3: Framework (FastAPI/NestJS/Spring Boot)
- Month 3-4: Database design and SQL
- Month 4-5: API design (REST/GraphQL)
- Month 5-6: Authentication and testing
- Month 6-12: Build production projects

### DevOps Engineer Path (1-2 years)
- Month 1-2: Linux fundamentals and bash scripting
- Month 2-4: Docker and containerization
- Month 4-6: Kubernetes basics and clusters
- Month 6-9: Terraform and Infrastructure as Code
- Month 9-12: CI/CD pipeline design
- Month 12+: Cloud platform specialization (AWS/GCP/Azure)

## Tips

- **Be honest about your experience** - Roadmap accuracy depends on this
- **Set a realistic time commitment** - Consistency beats intensity
- **Build projects** - Theory alone isn't enough
- **Don't jump paths** - Master one thoroughly first
- **Get feedback** - Share projects with community

## Next Steps After Learning Path

1. Follow the customized roadmap
2. Use Agent consultations for specific questions
3. Build projects from the suggested ideas
4. Join communities (Discord, Reddit, GitHub)
5. Share progress and get feedback
6. Apply for roles with portfolio projects

---

Once you select a learning path, you'll be matched with the most relevant agent who can provide detailed guidance for each step of your journey.
