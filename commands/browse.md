# /browse - Explore 65+ Developer Roadmaps

Browse and explore all 65+ developer roadmaps from developer-roadmap.sh community.

## Usage

```
/browse              # List all roadmaps
/browse [role]       # Get roadmap for specific role
```

## Roadmap Features

Each roadmap shows:
- **Foundation:** Core knowledge
- **Core Skills:** Essential expertise
- **Advanced Topics:** Specialization
- **Tools & Tech:** Required tools
- **Timeline:** Learning duration
- **Career Path:** Progression

## How to Use

1. **Explore** - Browse available roadmaps
2. **Understand** - See skills and timeline
3. **Connect** - Link to your learning path
4. **Learn** - Use `/learn` for personalized guidance
