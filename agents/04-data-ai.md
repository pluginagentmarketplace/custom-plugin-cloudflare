# Data & AI

## Career Paths

### Data Engineer
**Timeline:** 1-2 years | **Stack:** Python/Scala + Spark + Airflow + PostgreSQL/Snowflake

```
SQL Mastery → Python → ETL/Pipelines → Big Data (Spark) → Data Warehouse → Orchestration
```

**Key Skills:** SQL, Data pipelines, ETL/ELT, Performance tuning

---

### Machine Learning Engineer
**Timeline:** 1-2 years | **Stack:** Python + scikit-learn/TensorFlow + MLflow

```
Python → Math (Linear Algebra, Calculus) → ML Algorithms → Deep Learning → Production
```

**Key Skills:** Algorithms, Feature engineering, Model evaluation, Optimization

---

### AI Engineer (Emerging)
**Timeline:** 6-12 months | **Stack:** Python + LLM APIs + LangChain

```
LLM Fundamentals → Prompt Engineering → LangChain → AI Agents → Production Deployment
```

**Key Skills:** Prompt engineering, AI agents, RAG, Evaluation

## ML Workflow

```
Problem Definition → Data Collection → Preprocessing → EDA → Feature Engineering
        ↓
Model Selection → Training → Evaluation → Hyperparameter Tuning
        ↓
Validation → Deployment → Monitoring → Retraining
```

## Key Algorithms

**Supervised Learning:** Linear Regression, SVM, Decision Trees, Random Forest, Gradient Boosting

**Unsupervised:** K-Means, PCA, Clustering

**Deep Learning:** Neural Networks, CNNs, RNNs, Transformers

**Generative:** LLMs, Diffusion Models
