# Core Development Paths

## Quick Decision Matrix

| Path | Duration | Best For | Tech Stack |
|------|----------|----------|-----------|
| **Frontend** | 3-6 mo | UI/UX focus | React/Vue + JavaScript |
| **Backend** | 6-12 mo | Logic/APIs | Python/Node + Database |
| **Full Stack** | 6+ mo | Complete apps | Next.js + PostgreSQL |
| **DevOps** | 1-2 yr | Infrastructure | Docker + Kubernetes |

## Frontend Learning Path

```
HTML5 & CSS3 → JavaScript ES6+ → React (or Vue/Angular) → State Mgmt → Testing → Deploy
```

**Key Skills:** Responsive design, Components, Hooks, Routing, API integration, Performance optimization

**Timeline:** 3-6 months to entry-level

---

## Backend Learning Path

```
Language (Python/JS/Java) → Framework → Database → API Design → Auth → Testing → DevOps
```

**Popular Stacks:**
- Python: FastAPI + PostgreSQL
- JavaScript: NestJS + PostgreSQL
- Java: Spring Boot + PostgreSQL
- Go: Gin + PostgreSQL

**Timeline:** 6-12 months to entry-level

---

## Full Stack Learning Path

```
Frontend Mastery + Backend Mastery + Integration + DevOps + Scaling
```

**Recommended:** Next.js (React fullstack) + PostgreSQL + Vercel

**Timeline:** 6+ months to junior full-stack

---

## DevOps Learning Path

```
Linux & Bash → Docker → Kubernetes → Cloud (AWS/GCP) → Terraform → Monitoring
```

**Skills:** System administration, containerization, orchestration, infrastructure automation

**Timeline:** 1-2 years (requires prior dev/sysadmin experience)

---

## Technology Recommendations

**Frontend:**
- Framework: React (largest) or Vue (easiest)
- Styling: Tailwind CSS
- State: Zustand or Redux
- Testing: Vitest + Playwright

**Backend:**
- Language: Python (easiest) or JavaScript
- Framework: FastAPI or NestJS
- Database: PostgreSQL
- Auth: JWT or OAuth2

**Full Stack:**
- Meta-framework: Next.js
- Database: PostgreSQL
- Auth: Built-in or NextAuth.js
- Deploy: Vercel

**DevOps:**
- Container: Docker
- Orchestration: Kubernetes
- IaC: Terraform
- Cloud: AWS (most popular)
